Input Explanation

Input consists of N line(s) for each sites. Each line consists of two integer for site's x-coordinate and y-coordinate.

Example :

250 250
125 400
400 300

The input above will give 5 sites { (150,150), (250, 250), (125,400), (400,300), (375,125) } for program to generate the diagram.
